#  My Clock 🕒

A beautiful, smooth Analogue Clock with digital display built with HTML, CSS, and JavaScript.

Clock Preview 👉 ( https://iadeniyi056-byte.github.io/Analogue-Clock/)

 ✨ Features

- Real-time Analogue Clock with smooth moving hands
- Digital Time Display (HH:MM:SS)
- Current Date shown on the clock face
- Elegant Dark Theme design
- Responsive (works well on mobile and desktop)
- Clean and minimal UI

## 🛠️ Technologies Used

- HTML5
- CSS3 (Custom styling + Flexbox)
- Vanilla JavaScript (Date object + `setInterval`)
- Bootstrap Icons (for settings icon)

## 🚀 Features Explained

- The three hands (hour, minute, second) move accurately based on real time
- Smooth transitions on the hands
- Digital time updates every second
- Fully functional without any external dependencies

## How to Use

1. Download or clone the project
2. Open `index.html` in any browser
3. Watch time fly! ⏰
